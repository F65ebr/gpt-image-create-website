# gpt-image-create-website

一个调用 `gpt-image-2` 模型的纯前端图像生成页面，单 HTML 文件，无后端依赖，可直接通过 GitHub Pages 部署访问。

## 在线访问

部署后访问：<https://f65ebr.github.io/gpt-image-create-website/>

首次使用需在页面右上角「设置 API」中填写自己的 API Base URL 和 Token，配置存储在浏览器本地，不会上传。

## 功能

- 文本生成图像，支持多种宽高比与分辨率（最高 4K）
- 上传 / 拖拽 / 粘贴参考图，进行图像编辑
- 可调质量、输出格式（PNG/JPEG/WebP）、透明背景、内容审核等高级参数
- 一键下载生成结果
